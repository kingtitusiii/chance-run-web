# Chance Run — Web Build

Play in your browser: **https://kingtitusiii.github.io/chance-run-web/**

A wholesome pixel-art "help your town" game. Walk the streets, talk to all 35
villagers, carry parcels, and watch the whole town bloom because of you.

## What's in the town

- **Deliveries** — pick up at bakeries (gold beacons), drop at porches, or
  hold a throw and swish the mailbox for a bonus
- **Neighbors in need** — pink beacons: runaway puppies, spilled groceries,
  lost kids, fire drills, story time at the school
- **40+ story quests** — multi-step chains with town mysteries (who *is* the
  Midnight Gardener? whose letters were in the lighthouse?)
- **Living villagers** — schedules, jobs, gossip, birthdays, best friends,
  gifts, and letters in your mailbox
- **Seasons** — a 7-day cycle through spring, summer, autumn, winter with
  its own weather, fish, festivals, ambience, and stall specials
- **Market day** every 7th day: extra stalls, delivery tips
- **Cozy verbs** — fish (10 species + one golden rumor), stargaze, garden,
  pet everything, ride Jonas's bike (ring the bell), take photos
- **Make it yours** — pick a name in Profile (email or wallet sign-in
  optional), climb three leaderboards, and save your snapped photos as
  real PNGs from the Memories screen

## Controls

| | Keyboard | Touch |
|---|---|---|
| Walk | WASD | left stick |
| Talk / help / fish / pet | F | TALK |
| Throw (hold to aim) | R | THROW |
| Track a mission | Tab | TRACK |
| Quest board | Q | QUESTS |
| Map / photo | M / P | — |
| Pause & menus | Esc | II |

Saves persist in your browser, and the **Save data ⇄** screen exports your
town to the clipboard so you can import it on another device. Options include
master/music volume, fullscreen, reduced motion, large text, relaxed timers,
and quick days. The game auto-pauses when you tab away.

Built with Unity 6 (v0.2.0). This repo holds only the built game; the source
lives in a private repository. Desktop (Steam-ready) builds exist for macOS
and Windows.
